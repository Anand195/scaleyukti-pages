Why Hetzner CPX41? At €29.54/mo for the EU node (or €49/mo for Singapore), it offers 8 AMD EPYC vCPUs and 16GB RAM at 3–4× lower cost than DigitalOcean or Vultr's equivalent. The CPX41 scores 6,192 multi-core Geekbench6 — 2.2× faster than a Dell Precision baseline. For CA SmartDoc's OCR pipeline (Tesseract/EasyOCR), RAG embedding (sentence transformers), and LLM inference (Ollama/llama.cpp), 16GB RAM is the minimum viable spec and this plan delivers it at the lowest verified price. Singapore datacenter gives ~60–90ms latency to Ahmedabad — acceptable for a web interface. EU datacenter + VPN is also an option for cost-sensitive early deployments.

🇮🇳 India Latency Note — Important for CA Firm Demos

Hetzner's nearest datacenter to India is Singapore (sin-dc1) — average latency 60–90ms to Ahmedabad. For a document upload + extraction workflow, this is acceptable (users won't notice). For real-time AI chat, responses will feel slightly delayed. Recommendation: Start with EU for cost, switch to Singapore when onboarding first 3 clients. Hetzner lets you migrate with one click. Alternative: GigaNodes runs AMD EPYC in India at ₹1,800/mo but support is less established.

Scaling Strategy
When to scale up (vertical) vs scale out (horizontal) — decision framework for CA SmartDoc
↑
Vertical Scaling
Upgrade the same server to larger specs
Use when
  • OCR processing queue is backing up (CPU-bound)
  • LLM inference is slow — swap to larger model (needs 32GB+ RAM)
  • Single-firm deployment — one server serves one client
  • You have fewer than 5 active client firms
Hetzner vertical path
  • CPX41 (16GB, €29/mo) → CPX51 (32GB, €58/mo) → CX52 (32GB dedicated, €60/mo)
  • One-click resize in Hetzner console — zero downtime with snapshots
Rule: Go vertical first if RAM or CPU is the bottleneck on a single-tenant deployment.
→→
Horizontal Scaling
Add more servers, distribute load
Use when
  • 10+ active CA firm clients (each needing isolated data)
  • You need client data isolation (separate server per client)
  • High availability required — single server failure unacceptable
  • Processing volume exceeds single server capacity
Horizontal architecture
  • OCR worker nodes (stateless) — scale independently from AI/RAG layer
  • Shared vector database on separate node (Qdrant/Weaviate)
  • Load balancer (Hetzner's built-in, €6/mo) routes requests
  • Each client firm gets isolated data namespace, not isolated server
Rule: Horizontal when you have 10+ clients. Start vertical — infrastructure stays simple and cost-controlled.
Phase-by-Phase Infrastructure Roadmap
Based on CA SmartDoc pricing tiers (Starter ₹2,999 / Professional ₹5,999 / Enterprise ₹12,999)
Phase 1 · Month 0–3 · 1–5 clients
Single Hetzner CPX41
One EU or Singapore server. All services on single node: OCR (Tesseract), embeddings (sentence-transformers), vector store (Qdrant), LLM (Ollama + llama3.1:8b), API (FastAPI). Client data isolated via namespaces.
Cost: ₹2,700–₹4,500/mo · Break-even at 1 Starter client
Phase 2 · Month 3–9 · 5–15 clients
Vertical upgrade + backup
Upgrade to CPX51 (32GB) to run larger LLM (llama3.1:70b quantised) or serve multiple clients simultaneously. Add automated daily snapshot backup. Add monitoring (Uptime Kuma, Grafana). Still single server.
Cost: ~₹5,500/mo · Break-even at 2 Starter or 1 Professional client
Phase 3 · Month 9–18 · 15–40 clients
Separate OCR + RAG layers
Split OCR processing to dedicated worker server (CPX31, 8GB). Main server handles RAG queries and LLM. Shared Qdrant on separate small node. Hetzner Load Balancer (€6/mo) routes traffic. Each client has isolated vector collection.
Cost: ~₹10,000–₹12,000/mo · 4+ Professional clients cover this
Phase 4 · Month 18+ · 40+ clients
Multi-region + dedicated servers
India-dedicated node (GigaNodes or DigitalOcean Bangalore) for latency-sensitive enterprise clients. EU/SG nodes for cost-optimised clients. Enterprise clients get dedicated server (Hetzner AX41, AMD Ryzen, 64GB, €55/mo). Kubernetes optional at this scale.
Cost: ₹25,000–₹50,000/mo · 4+ Enterprise clients or 15+ Professional clients

✅ First-Server Setup Checklist (Hetzner CPX41)

✓ Create Hetzner account — console.hetzner.com (no setup fee)
✓ Select CPX41 plan · Ubuntu 22.04 LTS · Singapore DC (sin-dc1)
✓ Add SSH key before creation — never use password login
✓ Assign floating IP (€1.19/mo) for stable public address
✓ Enable Hetzner Firewall — block all except 22 (SSH), 80, 443, 8000 (API)
✓ Install Docker + Docker Compose (one-click app available)
✓ Deploy: Qdrant (vector DB), Ollama (LLM), FastAPI (backend), Nginx (proxy)
✓ Set up automated snapshots (€0.0119/GB/mo) — daily schedule
✓ Install Uptime Kuma for health monitoring (free, self-hosted)
✓ Test OCR pipeline on 50 sample invoices before first client demo
✓ Set RAM alert at 80% — upgrade to CPX51 when consistently exceeded
✓ Document all env vars — use .env file with Vault for secrets
