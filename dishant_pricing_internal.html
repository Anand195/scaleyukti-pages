Critical Assumptions
What This Pricing Is Built On

These assumptions must hold for the margin to work. Flag immediately if any change.

  • A1
    Dishant uses their own Genspark Enterprise API key Monthly AI API costs (Genspark/OpenAI usage) are billed directly to Dishant's account — NOT ours. If we absorb API costs, ₹8,000/month is insufficient. Minimum would need to be ₹12,000-15,000. CONFIRM before contract signing.
  • A2
    Existing VPS infrastructure — no new server setup needed Dishant already has Docker + n8n running. If not, server procurement + setup adds ₹5,000 one-time + ₹2,000-3,000/month ongoing. Adjust pricing accordingly.
  • A3
    Employeewise docs are complete and in digital format If docs are incomplete, we spend extra time on KB audit and gap-filling. Cap extra work at 8 hours — beyond that, scope change billable at ₹1,500/hr.
  • A4
    Genspark API available within 48 hours of kickoff If Genspark API is delayed and we must use OpenAI direct fallback — no cost change, same quality. But Dishant must get an OpenAI API key instead. Not a blocker, just document the swap.
  • A5
    No major scope change during development Phase 1 scope is fixed. If Dishant adds features mid-build (e.g., group messaging, image handling, multi-language) — these are Phase 1.5 and billed separately. Set this expectation in the kickoff call.

Pricing Risks
What Could Hurt Margin

🚨 Genspark API on ScaleYukti Account (not Dishant's)

If Dishant can't get their own Genspark API key and we run on our OpenAI account, monthly API costs at 50 employees active usage = ₹4,000-7,000/month. At ₹8,000 retainer, we make ₹1,000-4,000 before our maintenance time. Not viable.

✅ Fix: Contract must specify API account responsibility. Dishant provides their own key. If they can't, retainer goes to ₹14,000/month minimum.

⚠️ Scope Creep During Development

"Can we also add group message support?" / "Can the bot handle images?" / "Can we add a web dashboard?" — common asks once client sees the bot working. Each of these is 3-5 days of work.

✅ Fix: Phase 1 scope is locked in contract. Written change request + quote for anything outside. No verbal commitments.

⚠️ Credentials Delayed Beyond 1 Week

If Dishant delays credential handover past 7 days from agreement signing, our developer time is blocked. At ₹1,500/hr opportunity cost, 1 week delay = ₹45,000-60,000 in opportunity cost.

✅ Fix: Contract clause — "14-day timeline begins on Day 0 (credential handover). Delays on Dishant's side extend the deadline day-for-day. No refund for delays caused by client."
